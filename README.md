# 🎯 TaskTracker

Simple TO-DO code-along Application, it uses local-storage to save your TO-DOS!

## 🚀 Demo
👉 [See on GH Pages](https://Mattewww.github.io/tasktracker)

## 📦 Technologies
- HTML & CSS
- JavaScript
- React 19
- Vite

## 🛠️ Instalation 

   ```sh
   git clone https://github.com/Mattewww/tasktracker.git
   cd tasktracker
   npm install
   npm run dev
   ```
